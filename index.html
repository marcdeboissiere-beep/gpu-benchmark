⚡ About CPU-offloaded results (amber values)

When a model is larger than your GPU's VRAM, Ollama automatically runs some layers on your CPU. These results are shown in amber — they're valid but represent a different performance profile.

What affects speed in offload mode:

  • System RAM bandwidth — DDR5-6000 vs DDR4-3200 can double inference speed
  • PCIe bandwidth — Data transfers between GPU and RAM (PCIe 4.0 x16 = 32 GB/s)
  • % of model on GPU — More VRAM = more layers on fast GPU = faster overall
  • CPU core count — Less important than RAM bandwidth for transformers

Is it usable? At 2–8 tok/s, many users find it acceptable for chat and creative writing — about the speed of fast human reading. Not suitable for rapid iteration or long document processing.

Methodology

Benchmarks run via Ollama with fixed prompts and fixed token generation length. Rankings use median tokens/second and include variability metrics; reliability fields use structured run status events when available.

Results are aggregated by GPU model name and may vary by CPU, RAM, motherboard, cooling, driver version, and system load.
